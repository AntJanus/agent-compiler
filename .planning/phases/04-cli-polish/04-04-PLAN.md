Create src/cli/commands/compile.ts:

1. Imports (use CORRECT function names and types):

   ```typescript
   import { readFile } from 'node:fs/promises';
   import { intro, outro, log, spinner } from '@clack/prompts';

   import { parseSkillFile } from '../../parser/index.js';
   import { parseCommandFile } from '../../parser/index.js';
   import { mergeEmbeddedContent } from '../../embedding/index.js';
   import { selectTargetFile, selectContentTypes, selectSkills, selectCommands } from '../prompts/index.js';
   import { withSpinner, ActionableError } from '../output/index.js';
   import type { ParsedSkill, ParsedCommand, DiscoveredFile } from '../../types/index.js';
   ```

2. Export interface CompileOptions:
   ```typescript
   export interface CompileOptions {
     dryRun?: boolean;
     cwd?: string;
   }
   ```

3. Export async function runCompile(options: CompileOptions = {}): Promise<void>

4. Implementation flow:
   a. Show intro banner: `intro(pc.bgCyan(pc.black(' agent-compiler ')))`

   b. Discover skills and commands with spinner:
      ```typescript
      const discovery = await withSpinner('Discovering skills and commands...', async () => {
        const result = await discoverAll({ cwd: options.cwd });
        return result;
      });
      ```

   c. Check if anything was found:
      - If no skills and no commands: log.warn('No skills or commands found'), outro, return

   d. Parse discovered skills - CORRECT API usage:
      - discovery.skills is DiscoveredFile[] (NOT string[])
      - Each DiscoveredFile has {path, location}
      - parseSkillFile requires (filepath, fileContent) - must read file first
      ```typescript
      const parsedSkills = await withSpinner('Parsing skills...', async () => {
        const parsed: ParsedSkill[] = [];
        for (const skillFile of discovery.skills) {
          // Read file content first
          const fileContent = await readFile(skillFile.path, 'utf8');
          // parseSkillFile returns {metadata, content, warnings}
          const parseResult = parseSkillFile(skillFile.path, fileContent);
          // Convert to ParsedSkill structure
          parsed.push({
            path: skillFile.path,
            location: skillFile.location,
            metadata: parseResult.metadata,
            content: parseResult.content,
            referencedFiles: []  // TODO: extract references if needed
          });
        }
        return parsed;
      });
      ```

   e. Parse discovered commands (similar pattern with parseCommandFile)

   f. Prompt for target file:
      ```typescript
      const target = await selectTargetFile(options.cwd);
      if (!target) {
        outro(pc.yellow('Cancelled'));
        return;
      }
      ```

   g. Prompt for content types:
      ```typescript
      const contentTypes = await selectContentTypes();
      if (!contentTypes) {
        outro(pc.yellow('Cancelled'));
        return;
      }
      ```

   h. If contentTypes includes 'skills' and parsedSkills.length > 0:
      - Prompt skill selection
      - Handle cancellation

   i. If contentTypes includes 'commands' and parsedCommands.length > 0:
      - Prompt command selection
      - Handle cancellation

   j. Handle dry-run mode (NO TYPE CONVERSION - use ParsedSkill directly):
      ```typescript
      if (options.dryRun) {
        log.info(pc.yellow('Dry run mode - no files will be modified'));
        log.info(`Target: ${target.path}`);
        log.info(`Skills: ${selectedSkills.length}`);
        log.info(`Commands: ${selectedCommands.length}`);
        // Show what would be embedded
        for (const skill of selectedSkills) {
          log.info(`  - ${skill.metadata.name} (${skill.location})`);
        }
        outro(pc.yellow('Dry run complete - no changes made'));
        return;
      }
      ```

   k. Perform embedding with spinner - mergeEmbeddedContent expects ParsedSkill[] directly:
      ```typescript
      const result = await withSpinner('Embedding content...', async () => {
        return await mergeEmbeddedContent({
          targetPath: target.path,
          skills: selectedSkills,      // ParsedSkill[] - NOT Skill[]
          commands: selectedCommands,  // ParsedCommand[] - NOT Command[]
          backupDir: '.agent-compiler-backups'
        });
      });
      ```

   m. Show success summary:
      ```typescript
      if (result.skipped) {
        log.info('Content unchanged - skipped write');
      } else {
        log.success(`Embedded ${selectedSkills.length} skills and ${selectedCommands.length} commands`);
        if (result.backupPath) {
          log.info(`Backup created: ${result.backupPath}`);
        }
      }
      outro(pc.green('Done!'));
      ```

5. Wrap entire function body in try/catch:
   - If ActionableError, console.error(error.format())
   - Otherwise, console.error(pc.red(error.message))
   - Set process.exitCode = 1

CRITICAL API NOTES:
- discoverAll() returns {skills: DiscoveredFile[], commands: DiscoveredFile[]}
- DiscoveredFile has {path: string, location: 'global' | 'project'}
- parseSkillFile(filepath, fileContent) - must pass file content, NOT just path
- mergeEmbeddedContent expects ParsedSkill[] and ParsedCommand[] directly
- Do NOT convert ParsedSkill to Skill - that's wrong
